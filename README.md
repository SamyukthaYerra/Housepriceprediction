# House Price Prediction using Machine Learning

This project focuses on predicting residential house prices based on various features such as location, square footage, and structural attributes using machine learning regression techniques. 

The entire analysis and model development were conducted efficiently using **Google Colab**.

## 🚀 Open in Google Colab
You can run this notebook directly in Google Colab by uploading the repository files or linking your GitHub account.

## 📂 Project Structure
* `House_Price_Prediction (1).ipynb` - Jupyter Notebook containing data preprocessing, Exploratory Data Analysis (EDA), feature engineering, model training, and evaluation.
* `House Price Prediction Dataset.csv` - The target dataset containing housing market values and property details.
* `README.md` - Documentation of the project workflow.

## ⚙️ Google Colab Setup & Workflow
1. **Environment:** Executed on cloud-hosted Jupyter environment using Google Colab's hardware accelerators.
2. **Data Ingestion:** Uploaded `House Price Prediction Dataset.csv` directly into the Colab session storage or via Google Drive mount.
3. **Exploratory Data Analysis (EDA):** Visualizing correlations between physical house metrics and market price values using Seaborn and Matplotlib.
4. **Model Selection:** Implemented Regression algorithms (such as Linear Regression / Random Forest) using Scikit-Learn.
5. **Performance Evaluation:** Validated accuracy using metrics like R-squared score and Mean Absolute Error (MAE).
